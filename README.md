> This project applies NLP to 15 real BRSR filings from Indian listed companies (FY2022–25), quantifying how *specific* and *verifiable* their ESG claims are across five sectors.

---

## What is BRSR?

India's **Business Responsibility and Sustainability Report (BRSR)** is mandated by SEBI for the top 1,000 listed companies from FY2022–23. Unlike voluntary CSR reports, BRSRs follow a standardised 9-principle template covering environmental, social, and governance (ESG) topics under the National Guidelines on Responsible Business Conduct (NGRBC). This creates a rare, nationally standardised, publicly accessible ESG corpus — ideal for systematic text analysis.

## Methodology

The pipeline computes five NLP metrics per report:

```
Specificity Ratio (SR)         → % of sentences with a quantified claim (number + unit)
LM Finance Sentiment           → Loughran-McDonald (2011) positive/negative word density
Action-Vague Verb Ratio (AVVR) → Action verbs (achieved/certified) vs. aspirational (aims/seeks)
Boilerplate Density (BD)       → % text matching known BRSR stock phrases
Flesch Reading Ease (FRE)      → Text complexity

Composite Greenwashing Risk Score (GRS):
  GRS = (1−SR)×40 + (1−AVVR)×25 + BD×20 + LM_neg×15
  Tiers: LOW < 30 | MEDIUM 30–60 | HIGH > 60
```

**FinBERT** (Araci, 2019) sentence-level scoring is available in [`notebooks/FinBERT_Colab.ipynb`](notebooks/FinBERT_Colab.ipynb) — run it free on Google Colab (requires GPU for speed).

## Honest Limitations (read before citing)

1. **n = 15 is small.** Kruskal-Wallis p-values are non-significant (p > 0.05). Sector differences are descriptive only.
2. **8 of 15 reports use structured extracts**, not full-PDF text, due to domain-level robots.txt blocks. SR scores for those reports are likely underestimated.
3. **GRS weights are theoretically motivated**, not empirically fitted to labelled data.
4. **FinBERT not integrated live** — run the Colab notebook separately.
5. **GRS ≠ actual greenwashing** — it measures textual credibility risk, not performance vs. claims.

These limitations are fully documented in the [research report](reports/BRSR_Research_Report.md).

## How to Reproduce

```bash
# 1. Clone the repo
git clone https://github.com/rupalrani/brsr-nlp-greenwashing.git
cd brsr-nlp-greenwashing

# 2. Install dependencies
pip install nltk textstat openpyxl pandas numpy matplotlib seaborn scikit-learn scipy

# 3. Download NLTK data (one-time)
python -c "import nltk; nltk.download(['punkt','stopwords','wordnet','averaged_perceptron_tagger'])"

# 4. Run the full pipeline
python src/nlp_pipeline.py

# 5. Outputs land in outputs/
# 6. For FinBERT: open notebooks/FinBERT_Colab.ipynb in Google Colab
```

All 15 corpus files are in `data/raw/`. Source URLs and extraction notes are documented inside each `.txt` file header.
